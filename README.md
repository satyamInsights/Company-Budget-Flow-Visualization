# 📊 Company Budget Flow Visualization

## Overview

This project demonstrates how interactive visualizations can be used to understand company budget allocation across departments.

The project uses:

* Python
* Pandas
* Plotly Graph Objects
* Plotly Express

## Visualizations

### 1. Sankey Diagram

Displays the flow of company budget from the organization to departments and then to individual sub-departments.

Features:

* Interactive Hover
* Budget Flow
* Percentage Contribution

### 2. Sunburst Chart

Represents the hierarchical relationship between departments and sub-departments.

Features:

* Interactive Drill Down
* Budget Distribution
* Hierarchical Analysis
